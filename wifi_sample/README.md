# Arduino UNO R4 Wi-Fi UDP 通信サンプル

2 台の **Arduino UNO R4 WiFi** を使って **UDP 通信** を行うサンプルです。  
1 台は **アクセスポイント (AP)**、もう 1 台は **クライアント** として動作します。

## 設定

- SSID: `arduino`
- password: `password`
- port: `2390`

## 動作概要

- **アクセスポイント側**

  1.  アクセスポイントを起動
  1.  UDP ポート `2390` で通信を待機
  1.  受信したメッセージをシリアルモニタに表示

- **クライアント側**
  1.  アクセスポイントに接続
  1.  1 秒毎に UDP パケット `Hello world!` を アクセスポイント側 に送信
